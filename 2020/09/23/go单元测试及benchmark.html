Go单元测试及Benchmark

之前在刚开始写了如何编写测试程序。

使用断言：

go get -u github.com/stretchr/testify

Benchmark

文件名以下划线_benchmark结尾，方法名以Benchmark开头，参数为b *testing.B

在命令行输入 go test -bench=. -benchmem

Windows 下使⽤ go test 命令⾏时，-bench=.应写为-bench=”.”

使用 buffer 连接字符串的性能比 += 要好很多。
